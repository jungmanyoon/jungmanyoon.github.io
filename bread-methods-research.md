# 제과제빵 제법(Methods) 전문 자료

## 1. 스트레이트법 (Straight Dough Method / 직접법)

### 기본 특성
- **사전반죽 밀가루 비율**: 0% (사전반죽 없음)
- **사전반죽 수분 비율**: N/A
- **이스트 사용량**: 기준 (밀가루 대비 1.5-2.5%)
- **발효 시간**: 1-2시간 (1차 발효)
- **발효 온도**: 26-28°C
- **최종 반죽 조정**: 없음 (모든 재료 한 번에 투입)

### 풍미/식감 특성
- 단순하고 깨끗한 밀 풍미
- 부드럽고 균일한 크럼
- 제작 시간이 짧음
- 보존성: 보통 (1-2일)

---

## 2. 중종법 (Sponge Method)

### 기본 특성
- **사전반죽 밀가루 비율**: 40-70% (일반적으로 60%)
- **사전반죽 수분 비율**: 60-80% (베이커스 %)
- **이스트 사용량**: 기준 대비 70-80% (사전반죽에 전량 투입)
- **발효 시간**:
  - 1차(중종): 3-5시간
  - 2차(본반죽): 30-60분
- **발효 온도**: 24-26°C
- **최종 반죽 조정**:
  - 밀가루: 남은 40-30%
  - 물: 전체 수분량에서 중종 수분 제외
  - 소금: 본반죽에 투입
  - 유지류: 본반죽에 투입

### 풍미/식감 특성
- 복합적이고 깊은 풍미
- 부드럽고 촉촉한 크럼
- 향상된 볼륨
- 보존성: 우수 (3-4일)

---

## 3. 폴리쉬법 (Poolish)

### 기본 특성
- **사전반죽 밀가루 비율**: 20-40% (일반적으로 30%)
- **사전반죽 수분 비율**: 100% (1:1 비율)
- **이스트 사용량**: 기준 대비 50-60%
  - 폴리쉬: 밀가루 대비 0.1-0.3%
  - 본반죽: 추가 0.5-1%
- **발효 시간**:
  - 폴리쉬: 12-16시간 (냉장) 또는 3-5시간 (상온)
  - 본반죽: 1-2시간
- **발효 온도**:
  - 폴리쉬: 18-21°C (상온) 또는 4-6°C (냉장)
  - 본반죽: 26-28°C
- **최종 반죽 조정**:
  - 밀가루: 남은 70%
  - 물: 전체 수분량에서 폴리쉬 수분(밀가루 100%) 제외
  - 소금: 본반죽에 투입
  - 이스트: 추가 투입

### 풍미/식감 특성
- 복합적이고 약간 달콤한 풍미
- 큰 기공과 개방적인 크럼
- 얇고 바삭한 크러스트
- 보존성: 양호 (2-3일)

---

## 4. 비가법 (Biga)

### 기본 특성
- **사전반죽 밀가루 비율**: 30-50% (일반적으로 40%)
- **사전반죽 수분 비율**: 50-60% (단단한 반죽)
- **이스트 사용량**: 기준 대비 40-50%
  - 비가: 밀가루 대비 0.1-0.2%
  - 본반죽: 추가 0.5-1%
- **발효 시간**:
  - 비가: 12-18시간 (냉장) 또는 8-12시간 (상온)
  - 본반죽: 1.5-2.5시간
- **발효 온도**:
  - 비가: 16-18°C (상온) 또는 4-6°C (냉장)
  - 본반죽: 26-28°C
- **최종 반죽 조정**:
  - 밀가루: 남은 60%
  - 물: 전체 수분량에서 비가 수분(밀가루 50-60%) 제외
  - 소금: 본반죽에 투입
  - 이스트: 추가 투입

### 풍미/식감 특성
- 너티하고 복합적인 풍미
- 촘촘하고 쫄깃한 크럼
- 두껍고 바삭한 크러스트
- 보존성: 우수 (3-5일)

---

## 5. 르방법 (Levain/Sourdough)

### 기본 특성
- **사전반죽 밀가루 비율**: 15-30% (일반적으로 20%)
- **사전반죽 수분 비율**: 80-120% (스타터에 따라)
- **이스트 사용량**: 0% (자연 발효) 또는 기준 대비 20-30% (보조)
- **발효 시간**:
  - 르방: 4-8시간 (리프레시)
  - 본반죽: 3-6시간 (1차), 2-4시간 (2차)
- **발효 온도**:
  - 르방: 24-26°C
  - 본반죽: 24-26°C
- **최종 반죽 조정**:
  - 밀가루: 남은 80%
  - 물: 전체 수분량에서 르방 수분 제외
  - 소금: 본반죽에 투입
  - 이스트: 없음 또는 소량 보조

### 풍미/식감 특성
- 신맛이 있는 복합적 풍미
- 쫄깃하고 탄력 있는 크럼
- 두껍고 바삭한 크러스트
- 보존성: 매우 우수 (5-7일)

---

## 6. 저온발효 (Cold Fermentation)

### 기본 특성
- **적용 방식**: 모든 제법에 적용 가능
- **이스트 사용량**: 기준 대비 30-50% (발효 시간이 길어짐)
- **발효 시간**: 12-72시간
- **발효 온도**: 2-6°C (냉장고)
- **최종 반죽 조정**:
  - 이스트량 감소
  - 설탕량 약간 증가 (효모 활성화)
  - 복온 시간 필요 (1-2시간)

### 풍미/식감 특성
- 깊고 복합적인 풍미
- 향상된 크럼 구조
- 더 나은 소화성
- 작업 스케줄 유연성

---

## 7. 저온숙성 (Retarding)

### 기본 특성
- **적용 시점**: 성형 후
- **온도**: 2-6°C
- **시간**: 8-18시간
- **이스트 조정**: 기준량 유지 (발효는 거의 정지)
- **최종 반죽 조정**:
  - 복온 시간: 1-2시간
  - 최종 발효 시간 단축

### 풍미/식감 특성
- 풍미 향상
- 작업 일정 조절 가능
- 크러스트 색상 개선
- 오븐 스프링 향상

---

## 8. 오토리즈 (Autolyse)

### 기본 특성
- **사전반죽 밀가루 비율**: 100% (밀가루 + 물만)
- **사전반죽 수분 비율**: 최종 레시피와 동일
- **휴지 시간**: 20-60분 (일반적으로 30분)
- **온도**: 실온 (20-24°C)
- **이스트/소금**: 오토리즈 후 투입
- **최종 반죽 조정**:
  - 글루텐 발달로 믹싱 시간 20-30% 단축
  - 이스트량: 동일
  - 소금: 오토리즈 후 투입

### 풍미/식감 특성
- 향상된 신전성
- 더 나은 글루텐 구조
- 부드러운 크럼
- 믹싱 시간 단축

---

## 수화율(Hydration)에 따른 이스트 조정

### 기본 원칙
수화율이 높을수록 반죽이 부드러워 발효가 빨라지므로 이스트량 조정 필요

### 조정 공식
```
이스트 조정 계수 = 1 + (수화율 - 65%) × 0.002
```

### 예시
- **60% 수화율**: 기준량 × 0.9 (10% 감소)
- **65% 수화율**: 기준량 × 1.0 (기준)
- **70% 수화율**: 기준량 × 1.1 (10% 증가)
- **75% 수화율**: 기준량 × 1.2 (20% 증가)
- **80% 수화율**: 기준량 × 1.3 (30% 증가)

### 실무 가이드
| 수화율 범위 | 반죽 특성 | 이스트 조정 | 용도 |
|------------|----------|------------|------|
| 50-55% | 매우 단단함 | 기준 × 0.8 | 바게트, 하드롤 |
| 55-60% | 단단함 | 기준 × 0.9 | 식빵, 단과자빵 |
| 60-65% | 보통 | 기준 × 1.0 | 일반 식빵 |
| 65-70% | 부드러움 | 기준 × 1.1 | 치아바타 |
| 70-75% | 매우 부드러움 | 기준 × 1.2 | 포카치아 |
| 75-80%+ | 반죽처럼 흐름 | 기준 × 1.3 | 시아바타, 피자 |

## 온도에 따른 발효 시간 조정

### 기본 원칙
온도가 10°C 상승하면 발효 속도가 약 2배 증가

### 조정 공식 (효모 활성 범위: 4-40°C)
```
발효 속도 계수 = 2^((온도 - 26°C) / 10)
조정 시간 = 기준 시간 / 발효 속도 계수
```

### 온도별 발효 시간 가이드
| 온도 | 발효 속도 | 1시간 기준 → 실제 시간 | 비고 |
|------|----------|---------------------|------|
| 4°C | 0.24× | 1시간 → 4.2시간 | 냉장 발효 |
| 10°C | 0.33× | 1시간 → 3시간 | 저온 발효 |
| 16°C | 0.5× | 1시간 → 2시간 | 겨울철 실온 |
| 20°C | 0.7× | 1시간 → 1.4시간 | 봄/가을 실온 |
| 24°C | 0.87× | 1시간 → 1.15시간 | 이상적 실온 |
| 26°C | 1.0× | 1시간 → 1시간 | **기준 온도** |
| 28°C | 1.15× | 1시간 → 52분 | 여름철 실온 |
| 30°C | 1.32× | 1시간 → 45분 | 발효실 |
| 35°C | 1.87× | 1시간 → 32분 | 급속 발효 |
| 40°C | 2.64× | 1시간 → 23분 | 효모 활성 한계 |

### 실무 가이드
```
예시: 26°C에서 2시간 발효하는 레시피를 30°C에서 발효
- 발효 속도 계수 = 2^((30-26)/10) = 2^0.4 ≈ 1.32
- 조정 시간 = 2시간 / 1.32 ≈ 1.5시간 (90분)
```

### 온도 관리 주의사항
- **4°C 이하**: 발효 거의 정지 (저온숙성)
- **40°C 이상**: 효모 사멸 위험
- **최적 범위**: 24-28°C
- **냉장 발효**: 2-6°C, 12-72시간

## 소금이 발효에 미치는 영향

### 기본 원리
소금은 삼투압 작용으로 효모 세포에서 수분을 빼앗아 발효를 지연시킴

### 소금 농도에 따른 발효 지연
| 소금 비율 (밀가루 대비) | 발효 속도 영향 | 발효 시간 조정 | 용도 |
|----------------------|-------------|-------------|------|
| 0% | 기준 × 1.5 | 기준 × 0.67 | 오토리즈 단계 |
| 1.0% | 기준 × 1.1 | 기준 × 0.91 | 저염 빵 |
| 1.5% | 기준 × 1.0 | **기준** | 일반 빵 |
| 2.0% | 기준 × 0.85 | 기준 × 1.18 | 일반 빵 |
| 2.5% | 기준 × 0.7 | 기준 × 1.43 | 고염 빵 |
| 3.0% | 기준 × 0.5 | 기준 × 2.0 | 특수 빵 |

### 소금의 다양한 기능
1. **발효 조절**: 효모 활동 억제로 과발효 방지
2. **글루텐 강화**: 글루텐 구조 강화로 탄력성 향상
3. **풍미 향상**: 단맛과 대비되어 풍미 부각
4. **보존성**: 미생물 성장 억제로 보존 기간 연장

### 실무 적용
```
예시: 소금 1.5% → 2.5%로 증가 시
- 발효 속도: 0.7배로 감소
- 발효 시간: 1.43배로 증가
- 2시간 → 약 2시간 52분
```

### 소금 투입 시점 전략
1. **오토리즈 단계**: 소금 제외 (글루텐 발달 최적화)
2. **본반죽 초기**: 소금 투입 (일반적 방법)
3. **믹싱 후반**: 소금 투입 (글루텐 과도한 발달 방지)

## 종합 조정 공식

### 복합 요인 고려
실무에서는 여러 요인이 동시에 작용하므로 종합 조정 필요

```
최종 발효 시간 = 기준 시간 × 온도 계수 × 소금 계수 × 수화율 계수 × 제법 계수
```

### 예시 계산
**조건:**
- 기준: 26°C, 2시간, 소금 1.5%, 수화율 65%
- 실제: 30°C, 소금 2.5%, 수화율 75%, 폴리쉬법

**계산:**
- 온도 계수: 2^((30-26)/10) = 1.32
- 소금 계수: 0.7 (2.5% 소금)
- 수화율 계수: 1.1 (75% 수화율)
- 제법 계수: 0.8 (폴리쉬법 - 사전발효로 단축)

```
최종 시간 = 2시간 × (1/1.32) × 1.43 × 1.1 × 0.8
          = 2시간 × 0.76 × 1.43 × 1.1 × 0.8
          ≈ 1.9시간 (약 1시간 54분)
```

## 참고: 베이커스 퍼센티지 (Baker's Percentage)

### 기본 원칙
- 밀가루 = 100% (기준)
- 다른 재료 = 밀가루 대비 백분율

### 예시
```
밀가루: 1000g (100%)
물: 650g (65%) - 수화율
소금: 20g (2%)
이스트: 15g (1.5%)
설탕: 50g (5%)
```

### 사전반죽 계산 예시 (폴리쉬 30%)
```
전체 밀가루: 1000g (100%)
폴리쉬 밀가루: 300g (30%)
폴리쉬 물: 300g (100% 수화율)
본반죽 밀가루: 700g (70%)
본반죽 물: 650g - 300g = 350g
```

---

## 추가 참고사항

### 효모 종류별 전환 계수
- **생이스트 (Fresh)**: 기준 × 1.0
- **건조 이스트 (Active Dry)**: 기준 × 0.4
- **인스턴트 이스트 (Instant)**: 기준 × 0.33
- **자연 발효종 (Sourdough)**: 밀가루 대비 15-30%

### 설탕이 발효에 미치는 영향
- **0-5%**: 발효 촉진 (효모 영양)
- **5-10%**: 약간 지연 (삼투압)
- **10%+**: 발효 지연 (삼투압 증가)

### 우유/유제품의 영향
- 발효 속도 약간 지연 (지방 코팅)
- 풍미 향상
- 부드러운 크럼
- 갈색화 촉진 (유당)

## 실무 체크리스트

### 제법 선택 기준
1. **시간 여유**:
   - 짧음 → 스트레이트법
   - 보통 → 중종법, 폴리쉬
   - 충분 → 비가, 르방
2. **풍미 목표**:
   - 단순 → 스트레이트법
   - 복합 → 사전발효 제법
3. **보존 기간**:
   - 단기 → 스트레이트법
   - 장기 → 중종법, 비가, 르방

### 품질 관리 포인트
- ✅ 반죽 온도 26-28°C 유지
- ✅ 발효실 습도 75-80% 유지
- ✅ 발효 상태 확인 (손가락 테스트)
- ✅ 최종 발효 80-90% 완료 시 굽기
- ✅ 오븐 온도 정확도 확인
