# 제빵 제법 변환 알고리즘

## 1. 스트레이트법 (직접 반죽법)

### 개요
모든 재료를 한꺼번에 혼합하는 가장 기본적인 방법

### 특징
- 1차 발효: 실온에서 1-2시간
- 별도의 조정 없이 기본 레시피 그대로 사용
- 환경 요인에 따른 미세 조정 가능

## 2. 저온숙성법 (오버나이트법)

### 변환 공식

#### 이스트 감소율
- 인스턴트 이스트: 밀가루 중량의 0.8%
- 활성 건조 이스트: 밀가루 중량의 1.2%
- 생이스트: 밀가루 중량의 3%

#### 온도-시간 관계식
```
발효 시간 = 기본 시간 × 2^((표준 온도 - 실제 온도)/10)
```

### 실제 적용 예시
- 500g 밀가루 기준
- 표준 7.5g 이스트 → 3.75g으로 감소
- 반죽 후 22-25°C에서 마무리
- 4-6°C에서 12-24시간 발효

### 주의사항
- 4°C에서 이스트 활성은 실온의 10-20%로 감소
- 반죽 온도 관리가 핵심

## 3. 폴리쉬법 (Poolish)

### 핵심 원리
- **100% 가수율** (1:1 밀가루 대 물 비율)
- 액체 형태의 전발효종

### 변환 공식
```
폴리쉬 밀가루 % = (폴리쉬 밀가루 중량 / 전체 밀가루 중량) × 100
최종 반죽 물 = 전체 물 - 폴리쉬 물
최종 반죽 이스트 = 전체 이스트 - 폴리쉬 이스트
```

### 표준 배합
- 전체 밀가루의 20-30%를 폴리쉬로 사용
- 폴리쉬 밀가루 중량의 0.08-0.1% 이스트
- 20-22°C에서 12-18시간 발효

### 예시 (500g 밀가루, 25% 폴리쉬)
1. 폴리쉬: 125g 밀가루 + 125g 물 + 0.125g 이스트
2. 본반죽: 375g 밀가루 + 나머지 재료

## 4. 비가법 (Biga)

### 특징
- 50-60% 가수율의 단단한 전발효종
- 이탈리아식 제법

### 표준 배합
- 밀가루:물 = 100:50-60
- 이스트: 0.2-0.5%
- 12-24시간 상온 발효

## 5. 중종법 (스펀지법)

### 핵심 특징
- 밀가루의 50-70%를 전반죽에 사용
- **전체 이스트를 1차 반죽에 투입**
- 짧은 발효 시간 (30분-2시간)

### 배합 비율
- 밀가루:물 ≈ 100:60 (60% 수화율)
- 부재료 일부 포함 가능

### 적용 예시
- 1차 반죽: 밀가루 300g + 물 180g + 이스트 6g (전량)
- 2차 반죽: 나머지 재료 추가

## 6. 탕종법

### 과학적 원리
- 전분의 호화를 통한 수분 보유력 향상
- 1:5 비율 (밀가루:액체)로 65°C까지 가열

### 변환 공식
```
탕종 밀가루 = 전체 밀가루 × 0.08
탕종 물 = 탕종 밀가루 × 5
최종 반죽 물 = (전체 밀가루 × 0.75) - 탕종 물
수분 증발 보정 = +10-15g
```

### 효과
- 빵의 부드러움 증가
- 보존성 향상
- 촉촉한 식감 유지

## 7. 페이트 페르망테법 (구반죽법)

### 특징
- 소금까지 포함된 완전한 반죽을 발효종으로 사용
- 4-6시간 상온 발효 또는 냉장 숙성

### 표준 배합
- 밀가루:물 = 100:60
- 이스트 ≈ 1%
- 소금 ≈ 2%

## 💡 제법 선택 가이드

### 시간이 충분할 때
- 저온숙성법: 최상의 풍미와 질감
- 폴리쉬법: 부드럽고 풍미 있는 빵

### 시간이 부족할 때
- 스트레이트법: 가장 빠른 방법
- 중종법: 빠르면서도 품질 향상

### 특별한 질감을 원할 때
- 탕종법: 매우 부드럽고 촉촉한 빵
- 비가법: 쫄깃하고 구멍이 많은 빵

## 🔧 구현 시 고려사항

1. **정밀한 계산**: 소수점 둘째 자리까지 계산
2. **사용자 옵션**: 발효 시간, 온도 선택 가능
3. **자동 보정**: 환경 요인에 따른 미세 조정
4. **명확한 구분**: 1차/2차 반죽 재료 분리 표시
