# 팬 크기별 정밀 조정 시스템

## 범용 팬 크기 계산 공식

### 기본 공식
```
당신의 팬 부피 ÷ 레시피 팬 부피 = 재료 배수
```

### 부피 계산법
- **직사각형/정사각형 팬**: 부피 = 길이 × 너비 × 높이
- **원형 팬**: 부피 = π × r² × 높이

## 매직 넘버 시스템

### 핵심 공식
```
팬 부피(g) ÷ 매직 넘버 = 필요한 반죽 중량
```

### 빵 종류별 매직 넘버

| 빵 종류 | 매직 넘버 | 특징 |
|---------|-----------|------|
| 식빵 (흰 밀가루) | 1.78 | 표준 식빵용 |
| 통밀빵 | 1.33 | 밀도가 높아 더 많은 반죽 필요 |
| 부재료가 많은 빵 | 1.5-1.6 | 건포도, 견과류 등 포함 |

## 한국식 식빵팬 규격

### 표준 규격과 반죽량

| 규격 | 팬 크기 | 물 용량 | 필요 반죽량 |
|------|---------|---------|-------------|
| 1근(600g) | 약 19×9×9cm | 1600g | 600g |
| 1.5근(900g) | 약 20×10×12cm | 2400g | 900g |
| 2근(1200g) | 대형 상업용 | 3200g | 1200g |

### 실용적 계산법
```
팬 부피(g) ÷ 4 = 필요한 반죽 중량
```

## 굽기 시간과 온도 조정

### 팬 특성별 조정 규칙

| 팬 특성 | 온도 조정 | 시간 조정 |
|---------|-----------|-----------|
| 깊은 팬 | -25°F(-14°C) | +5-8분 |
| 얕은 팬 | +25°F | 시간 단축 |
| 어두운 색 팬 | -25°F | 변동 없음 |
| 유리 팬 | -25°F | 약간 증가 |

### 조정 이유
- **깊은 팬**: 열전달이 느려 낮은 온도로 긴 시간
- **어두운 팬**: 열 흡수가 빨라 온도를 낮춤
- **유리 팬**: 열전도가 느려 조정 필요

## 다중 팬 계산

### 여러 개 제작 시
```
총 필요 반죽량 = 팬당 반죽량 × 팬 개수
배합 배수 = 총 필요 반죽량 ÷ 기본 레시피 반죽량
```

### 예시
- 식빵 팬(842g 필요) 2개 제작
- 총 필요량: 1684g
- 기본 레시피(842g) → 2배 증량

## 제과용 팬 조정

### 케이크 팬 특징
- 팬 부피의 60-70%만 채움
- 팽창률이 빵과 다름

### 원형 케이크 팬 표준

| 직경 | 높이 | 권장 반죽량 |
|------|------|-------------|
| 15cm | 7cm | 400-450g |
| 18cm | 7cm | 600-650g |
| 21cm | 7cm | 800-900g |

## 커스텀 팬 대응

### 부피 측정 방법
1. 팬에 물을 가득 채움
2. 물의 무게(g) = 부피(ml)
3. 매직 넘버로 나누어 반죽량 계산

### 사용자 입력 옵션
- 팬 종류 선택 (드롭다운)
- 커스텀 부피 입력 (ml 단위)
- 빵 종류별 매직 넘버 자동 적용

## 실제 구현 예시

### 시스템 로직
```javascript
function calculateDoughWeight(panVolume, breadType) {
    const magicNumbers = {
        'white': 1.78,
        'whole_wheat': 1.33,
        'enriched': 1.6
    };
    
    return panVolume / magicNumbers[breadType];
}

function scaleRecipe(originalRecipe, targetWeight) {
    const scaleFactor = targetWeight / originalRecipe.totalWeight;
    
    return originalRecipe.ingredients.map(ingredient => ({
        ...ingredient,
        amount: ingredient.amount * scaleFactor
    }));
}
```

## 💡 사용자를 위한 팁

### 반죽량 확인 방법
- 반죽이 팬의 1/3 정도 차면 적정
- 최종 발효 후 팬 상단 1-2cm 아래까지 차야 함

### 팬 준비
- 버터나 오일로 도포
- 밀가루 살짝 뿌리기 (선택사항)
- 베이킹 페이퍼 사용 시 쉽게 분리

## 🔧 시스템 구현 시 고려사항

1. **팬 데이터베이스**: 주요 팬 규격 사전 입력
2. **비율 검증**: 극단적인 크기 변경 시 경고
3. **반올림 처리**: 실용적인 단위로 반올림
4. **시각적 피드백**: 팬 크기와 반죽량 그래픽 표시
