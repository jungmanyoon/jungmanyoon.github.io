# 환경 요인의 정밀한 보정

## 온도와 습도의 수학적 관계

### 온도 보정 공식

#### 발효 시간 조정
- **10°F(5.5°C) 상승마다**: 발효 시간 25-30% 감소
- **15°F(8°C) 하락마다**: 발효 시간 2배 증가

#### 계산 예시
```
조정된 발효 시간 = 기본 발효 시간 × 온도 계수

온도 계수 계산:
- 온도 상승 시: 0.75^(온도차이/10)
- 온도 하락 시: 2^(온도차이/15)
```

### 습도 보정 공식

#### 액체량 조정
- **상대습도 10% 증가마다**: 액체량 1-2큰술/컵 감소
- **습도 40% 이하일 때**: 밀가루 1컵당 액체 1-2큰술 증가

#### 실제 적용
```
조정 액체량 = 기본 액체량 + (습도 보정값)

습도 보정값:
- (40 - 현재습도) × 0.5 큰술 (습도 40% 이하)
- (현재습도 - 60) × -0.5 큰술 (습도 60% 이상)
```

## 원하는 반죽 온도(DDT) 공식

### 기본 DDT 공식

#### 전발효가 있는 경우
```
물 온도 = (4 × 원하는 반죽 온도) - 실온 - 밀가루 온도 - 전발효 온도 - 마찰 계수
```

#### 일반 반죽의 경우
```
물 온도 = (3 × 원하는 반죽 온도) - 실온 - 밀가루 온도 - 마찰 계수
```

### 마찰 계수

| 반죽 방법 | 마찰 계수 |
|-----------|-----------|
| 손반죽 | 5-8°F (3-4°C) |
| 스탠드 믹서 | 20-25°F (11-14°C) |
| 전문가용 나선형 믹서 | 15-20°F (8-11°C) |

### 계산 예시
- 원하는 반죽 온도: 78°F (25.5°C)
- 실온: 72°F (22°C)
- 밀가루 온도: 70°F (21°C)
- 스탠드 믹서 사용 (마찰계수 22°F)

```
물 온도 = (3 × 78) - 72 - 70 - 22 = 70°F (21°C)
```

## 고도별 정밀 조정

### 고도별 조정 표

| 고도 | 온도 조정 | 발효제 조정 | 액체 조정 | 설탕 조정 |
|------|-----------|-------------|-----------|-----------|
| 해수면-3,000ft | 변경 없음 | 변경 없음 | 변경 없음 | 변경 없음 |
| 3,000-5,000ft | +15-25°F | -1/8 tsp당 | +1-2 큰술 | -1-2 큰술/컵 |
| 5,000-7,000ft | +25°F | -1/8-1/4 tsp당 | +2-4 큰술 | -2-3 큰술/컵 |
| 7,000ft 이상 | +25-35°F | -1/4 tsp당 | +3-4 큰술 | -3 큰술/컵 |

### 고도 조정 이유

#### 낮은 기압의 영향
- 액체 증발 속도 증가
- 발효 속도 가속화
- 글루텐 구조 약화

#### 구조 강화 방법
- 밀가루 1-2큰술 추가
- 반죽 시간 증가
- 발효 시간 단축

## 계절별 조정 가이드

### 여름철 (고온다습)
- 이스트량 10-20% 감소
- 찬물 사용 (얼음 활용)
- 발효 시간 20-30% 단축
- 소금 약간 증량 (발효 억제)

### 겨울철 (저온건조)
- 이스트량 10-15% 증가
- 따뜻한 물 사용
- 발효 시간 연장
- 액체량 5-10% 증가

### 봄/가을 (적정 환경)
- 표준 레시피 그대로 사용
- 미세 조정만 필요

## 실시간 환경 모니터링

### 권장 측정 도구
- 디지털 온습도계
- 적외선 온도계 (반죽 온도)
- 고도계 (또는 GPS 활용)

### 자동 보정 시스템
```javascript
function calculateEnvironmentalAdjustments(recipe, environment) {
    const adjustments = {
        yeast: 1,
        liquid: 0,
        time: 1
    };
    
    // 온도 보정
    if (environment.temp > 85) {
        adjustments.yeast *= 0.8;
        adjustments.time *= 0.75;
    } else if (environment.temp < 65) {
        adjustments.yeast *= 1.2;
        adjustments.time *= 1.5;
    }
    
    // 습도 보정
    if (environment.humidity < 40) {
        adjustments.liquid += recipe.liquid * 0.05;
    } else if (environment.humidity > 70) {
        adjustments.liquid -= recipe.liquid * 0.03;
    }
    
    // 고도 보정
    if (environment.altitude > 3000) {
        adjustments.yeast *= 0.9;
        adjustments.liquid += recipe.liquid * 0.02;
    }
    
    return adjustments;
}
```
